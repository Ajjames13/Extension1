# Extension1
Chrome Extension (Manifest V3) scaffold built with React, TypeScript, and Vite.

## Folder structure

```
.
├── public
│   └── manifest.json
├── src
│   ├── dashboard
│   │   └── main.tsx
│   ├── popup
│   │   └── main.tsx
│   └── shared
│       ├── App.tsx
│       └── app.css
├── dashboard.html
├── popup.html
├── package.json
├── tsconfig.json
├── tsconfig.node.json
└── vite.config.ts
```

## Build instructions (load unpacked)

1. Install dependencies.
   ```bash
   npm install
   ```
2. Build the extension.
   ```bash
   npm run build
   ```
3. Open `chrome://extensions`, enable **Developer mode**, and click **Load unpacked**.
4. Select the `dist` folder generated by Vite.

## Notes

- `popup.html` is the extension action popup.
- `dashboard.html` is a secondary extension page you can link to from the popup.
- Shared React UI lives in `src/shared` and is reused in both pages.
